# 如何测量 square-size - 快速指南

## 📏 测量步骤（3步完成）

### 步骤1：准备工具
- ✅ 一把直尺或卷尺（精度越高越好）
- ✅ 你的棋盘格（打印的或显示的）

### 步骤2：测量方格
测量棋盘格中**一个完整方格**的边长：

```
┌─────────┬─────────┬─────────┐
│         │         │         │
│    A    │    B    │    C    │
│         │         │         │
├─────────┼─────────┼─────────┤
│         │         │         │
│    D    │    E    │    F    │  ← 测量这个方格的边长
│         │         │         │
└─────────┴─────────┴─────────┘
```

**测量位置**：
- 测量方格的**内边长**（不是外边框）
- 测量**多个方格**取平均值（更准确）

### 步骤3：输入数值
将测量结果（单位：毫米）输入到命令中：

```bash
# 如果测量结果是 25.0 毫米
python camera_calibration.py --square-size 25.0

# 如果测量结果是 20.5 毫米
python camera_calibration.py --square-size 20.5

# 如果不指定，程序会提示你输入
python camera_calibration.py
```

## 🎯 实际测量示例

### 示例1：打印的棋盘格

```
测量过程：
1. 在 A4 纸上打印了 9×6 棋盘格
2. 用尺子测量一个方格
3. 测量结果：24.8 mm
4. 再测量另一个方格：25.1 mm
5. 平均值：(24.8 + 25.1) / 2 = 24.95 mm

使用：
python camera_calibration.py --square-size 24.95
```

### 示例2：平板显示的棋盘格

```
测量过程：
1. 在 10 英寸平板上显示棋盘格
2. 用尺子测量屏幕上一个方格的实际物理尺寸
3. 测量结果：10.2 mm

使用：
python camera_calibration.py --square-size 10.2
```

## ⚠️ 常见错误

### ❌ 错误1：使用设计尺寸
```
错误：使用设计时的尺寸（如 25.0 mm）
正确：测量实际打印/显示后的尺寸
```

### ❌ 错误2：测量外边框
```
错误：测量整个棋盘格的外边框
正确：测量单个方格的内边长
```

### ❌ 错误3：单位错误
```
错误：输入 2.5（以为是厘米）
正确：输入 25.0（转换为毫米）
```

### ❌ 错误4：两个相机使用不同值
```
错误：
  相机1：--square-size 25.0
  相机2：--square-size 24.5

正确：
  相机1：--square-size 25.0
  相机2：--square-size 25.0  （必须相同！）
```

## ✅ 验证测量是否正确

标定完成后，检查：

1. **重投影误差**应该很小（< 0.5 像素）
2. **两个相机焦距**应该接近（差异 < 5%）
3. 如果误差很大，可能是 square-size 设置错误

## 💡 提示

- 📐 使用游标卡尺可以获得更高精度（0.1mm）
- 📊 测量多个方格取平均值，提高准确性
- 📝 记录你的测量结果，方便后续使用
- 🔄 如果打印时缩放了，必须重新测量
